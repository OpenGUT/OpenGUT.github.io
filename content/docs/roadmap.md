# Roadmap

## v0.2 (near-term)

- Wireless BLE streaming mode.
- Improved USB latency for low-lag live monitoring.
- Playground Windows installer.

## v0.3 (mid-term)

- On-device event detection (no host PC required).
- AudioSep denoising running on a local GPU without cloud dependency.
- MATLAB and Python SDK packages.

## v1.0 (long-term)

- Stable hardware revision with regulatory compliance documentation.
- Peer-reviewed dataset released under CC-BY.
- Fully open firmware with RTOS support.
